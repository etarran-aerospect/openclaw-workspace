- Working solution: use **mcporter** to manage MCP servers and connect Conway via **conway-terminal** (stdio).

- `conway-terminal` present (v2.0.9).
- Conway API key provisioned and stored locally:
  - `~/.conway/config.json` (chmod 600)
  - `~/.conway/` (chmod 700)
  - (Key value not to be echoed in chat.)
- Installed mcporter globally: `npm install -g mcporter@0.7.3`.
- mcporter home config created/updated: `~/.mcporter/mcporter.json` (chmod 600) with server:
  - name: `conway`
  - command: `conway-terminal`
  - env: `CONWAY_API_KEY` loaded from `~/.conway/config.json`

### Wrapper CLI

  - `conway-mcp call <tool> ...`
