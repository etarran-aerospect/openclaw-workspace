# Beriker Internal Proposal Package

Files:

- `beriker-internal-construction-proposal.md` - working internal estimate and scope.
- `phase-budget.csv` - phase cost table for spreadsheet editing.
- `beriker-labor-only-homeowner-proposal.md` - homeowner-facing labor-only proposal broken out by phase.
- `labor-only-phase-budget.csv` - labor-only phase table for spreadsheet editing.
- `proposal-print.html` - printable branded HTML version.
- `assets/` - Pacific Houseworks logo files.

Use the Markdown or HTML as the working version. Before sending to the client, remove internal pricing posture notes, internal risk notes, and the low/high strategy ranges.

Labor-only note: the homeowner-facing labor-only version includes Pacific Houseworks labor plus specialty trade labor allowances, but excludes materials, fixtures, appliances, permits, plan revisions, agency fees, special inspection fees, and owner selections unless separately added.
