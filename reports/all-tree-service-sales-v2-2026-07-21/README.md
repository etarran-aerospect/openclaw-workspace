# All Tree Service sales v2

This folder contains a sales-ready local business website mockup for All Tree Service.

Open `index.html` in a browser. Keep the `assets/` folder next to it so images load.

## Positioning

The pitch is not "you have no website." They already appear to have one.

The stronger pitch is:

- Cleaner mobile-first page for calls and texted job photos.
- Real job photos instead of generic contractor imagery.
- Clear Santa Rosa / Sonoma County service-area language.
- License number visible without making risky claims.
- No "Certified Arborist" wording.
- Faster path from visitor to estimate: call or text photos.

## Suggested offer

Start with a low-friction redesign offer:

> I put together a cleaner mobile version of your site focused on calls, texted job photos, real job pictures, and safer licensing language. If you like it, I can set it up on your domain and make it easy to update.

Pricing idea:

- $750 quick rebuild: one-page site, mobile-first, phone/text CTAs, basic SEO, existing assets.
- $1,200 better version: above plus review/gallery cleanup, analytics, contact form, hosting/domain help.
- $150/month optional care: edits, backups, small content updates, Google profile support.

## Notes

Do not claim they are certified arborists unless the owner provides proof. Keep the site focused on licensed tree service, trimming, removals, emergency cleanup, yard care, hauling, and commercial service.
